# 📌 Módulo: Python para Análise de Dados

Este módulo consolida os fundamentos da linguagem Python voltados para análise de dados, abrangendo desde conceitos introdutórios até manipulação de strings e estruturas de controle.

O conteúdo foi estruturado para desenvolver a base necessária para atuação em áreas como:

- Análise de Dados (Data Analysis)
- Engenharia de Dados (Data Engineering)
- Ciência de Dados (Data Science)
- Automação de Processos

A abordagem prioriza entendimento prático, com foco em sintaxe clara, tipos de dados, operadores, estruturas de controle e manipulação de strings.

---

# 🎯 Objetivo do Módulo

Capacitar o uso dos principais recursos da linguagem Python aplicados à análise e manipulação de dados, permitindo:

- Domínio da sintaxe básica e tipos de dados
- Utilização de operadores aritméticos, lógicos e de comparação
- Construção de estruturas condicionais e de repetição
- Manipulação eficiente de strings e dados textuais
- Preparação para módulos mais avançados (listas, dicionários, funções, etc.)

---

# 🗂️ Estrutura dos Módulos

O conteúdo está dividido em módulos progressivos, abordando desde os fundamentos da linguagem até manipulação de strings:

---

## 📁 1. Linguagem Python

Apresenta os fundamentos da linguagem Python:

- Tipos de dados (str, int, float, list, tuple, dict, set, bool)
- Variáveis e constantes (convenção)
- Regras de nomenclatura (snake_case)
- Conversão de tipos (type casting)
- Funções de entrada (input) e saída (print)

📌 **Objetivo:** compreender os conceitos básicos da linguagem.

👉 Acesse: `./1-Linguagem-Python`

---

## 📁 2. Operadores

Aborda os diferentes tipos de operadores em Python:

- Operadores aritméticos (+, -, *, /, //, %, **)
- Operadores de comparação (==, !=, <, >, <=, >=)
- Operadores de atribuição (=, +=, -=, *=, /=)
- Operadores lógicos (and, or, not)
- Operadores de identidade (is, is not)
- Operadores de associação (in, not in)

📌 **Objetivo:** dominar as operações e expressões em Python.

👉 Acesse: `./2-Operadores`

---

## 📁 3. Estruturas Condicionais e Repetição

Aborda controle de fluxo e estruturas de repetição:

- Indentação e blocos de código
- Estruturas condicionais (if, elif, else)
- Estruturas de repetição (for, while)
- Função range() para sequências numéricas
- Controle de loops (break, continue)

📌 **Objetivo:** controlar o fluxo de execução e implementar lógica dinâmica.

👉 Acesse: `./3-If-Else-For-While`

---

## 📁 4. Strings

Trabalha manipulação de dados textuais:

- Conceitos fundamentais (imutabilidade, indexação)
- Métodos de string (upper, lower, strip, split, join, replace)
- Interpolação de variáveis (%, format, f-strings)
- Fatiamento de strings (slicing)
- Strings de múltiplas linhas

📌 **Objetivo:** manipular e transformar dados textuais de forma eficiente.

👉 Acesse: `./4-Strings`
